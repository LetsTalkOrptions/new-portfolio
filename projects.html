Projects
See my work below. Unless stated otherwise, I am responsible for all aspects of the project. I am always working on something, so feel free to check back for awesome new content. All my code for the projects can be viewed through my GitHub.

MovieBase

MovieBase is a site I helped create in my BootCamp, August 2018. I was responsible for the concept, UI, all design, media queries, and usage of API's. It is an interactive site that uses AJAX calls to pull movie data from different APIs while displaying some movie facts. We wanted users to have a creative, clean, simple design that is attractive and compatible on any device. We also included a comments page where users could post reviews of movies and stored their responses in a Firebase.

All Aboard

All Aboard is a project I completed that is an updated look of a traditional train schedule board. It uses moment.js to display the current time and keep track of upcoming trains that users can input and add to the board while storing the data via Firebase. All time formulas are written in code and use moment.js when possible. Simple. Appealing. Functional.
